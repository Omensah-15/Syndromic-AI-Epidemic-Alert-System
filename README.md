## What It Does

SAEAS detects disease outbreaks in real-time by analyzing health reports with AI and environmental sensor data.

### Key Features
- **AI Risk Analysis**: Automatically analyzes health reports and predicts outbreak risks
- **Live Sensor Monitoring**: Connect Arduino ESP32 for real environmental data
- **Hotspot Mapping**: Visualize outbreak locations on interactive maps
- **Smart Alerts**: Get instant notifications for high-risk situations

## Quick Start

### Use Online (Recommended)
1. Visit the [live app](https://hpc7e9vyxqdcibx9vx8dkf.streamlit.app/)
2. Start submitting health reports immediately
3. No installation required - works in any browser

### With Arduino (Optional)
1. Upload provided code to ESP32
2. Connect sensors (temperature, humidity, water quality)
3. Plug USB into computer - app auto-detects within seconds

## How to Use

1. **Submit Reports**: Describe symptoms and cases in plain English
2. **View Analytics**: See risk scores, outbreak probabilities, and trends
3. **Monitor Map**: Watch real-time outbreak hotspots develop
4. **Get Alerts**: Receive immediate warnings for high-risk situations
